# BRRTRouter `auto-research/`

Autonomous and semi-autonomous **performance research** for the router: bounded experiments on an explicit **control surface**, with a **≥ 30 minute** gate (Tilt + lint + tests + benches) and **commit-forward** policy (no PRs for this track).

| Path | Role |
|------|------|
| [`docs/`](./docs/README.md) | Charter: control surface, experiment log, “won’t repeat”, cron checklist |
| [`scripts/`](./scripts/README.md) | Python helpers (`perf_iteration.py`) — no ad-hoc shell scripts for the same workflow |
