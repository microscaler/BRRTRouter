# llmwiki schema (BRRTRouter)

Conventions for [`./`](./):

| Location | Purpose |
|----------|---------|
| [`index.md`](./index.md) | Catalog — start here every session |
| [`log.md`](./log.md) | Append-only session notes (newest at bottom) |
| [`topics/`](./topics/) | Cross-cutting themes (performance, validation, generator, …) |

**End of session:** append [`log.md`](./log.md), update touched topic pages, fix any `> **Open:**` items you resolved.

See also: [Karpathy llm-wiki gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).
